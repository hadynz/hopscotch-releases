# Hopscotch Releases

This repository is used for hosting public releases of Hopscotch, as well as the changelog.

Hopscotch is **not** open source and this repo *DOES NOT* contain any source code for Hopscotch.

If you have a feature request or bug report, please go file an [issue](https://github.com/hadynz/hopscotch-releases/issues/new/choose).
